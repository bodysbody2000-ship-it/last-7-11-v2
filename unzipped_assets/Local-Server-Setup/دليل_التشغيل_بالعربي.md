# دليل التشغيل المحلي لمشروع مسابقة الكيمياء - مستر أحمد عبد الوهاب 🧪

هذا الدليل سيساعدك على تشغيل تطبيق **مسابقة الكيمياء التفاعلية** محلياً على جهاز الكمبيوتر الخاص بك بعد تنزيله من GitHub.

---

## المتطلبات الأساسية

تأكد من تثبيت البرامج التالية على جهازك أولاً:
1. **[Node.js](https://nodejs.org/)** (الإصدار 18 أو أحدث)
2. **مدير الحزم npm** (يتم تثبيته تلقائياً مع Node.js)
3. **[Git](https://git-scm.com/)** (لتنزيل المشروع من GitHub)

---

## 🚀 خطوات التشغيل على جهازك

### 1. تحميل كود المشروع
افتح terminal أو command prompt واكتب الأمر التالي لتنزيل المشروع:
```bash
git clone <رابط_مستودع_المشروع_من_جيت_هاب>
cd <اسم_مجلد_المشروع>
```

### 2. تثبيت الحزم والمكتبات (Dependencies)
قم بتثبيت جميع المكتبات البرمجية اللازمة لتشغيل الواجهة الأمامية والخلفية معاً:
```bash
npm install
```

### 3. إعداد ملف المتغيرات البيئية (Environment Variables)
قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع عبر نسخ الملف التجريبي:
```bash
cp .env.example .env
```
افتح ملف `.env` الجديد باستخدام أي محرّر نصوص، واضبط القيم التالية:
- `PORT`: المنفذ المفضل للتشغيل (مثال: `3000`)
- `SESSION_SECRET`: اكتب نصاً عشوائياً طويلاً لحماية جلسات المستخدمين (مثال: `chemistry_lab_secret_key_987`)

### 4. بدء تشغيل خادم التطوير (Development Server)
لتشغيل المشروع في وضع التطوير (والذي يتيح تحديث الصفحة تلقائياً عند تعديل أي ملف):
```bash
npm run dev
```
بعد تشغيل الخادم بنجاح، افتح متصفح الإنترنت واذهب إلى الرابط:
👉 **`http://localhost:3000`**

---

## 📦 التجهيز والتشغيل الفعلي (Production Build)

إذا كنت ترغب في تشغيل التطبيق بأعلى كفاءة وسرعة ممكنة للإنتاج الفعلي:

1. **بناء الملفات المحسنة:**
   ```bash
   npm run build
   ```
   سيقوم هذا الأمر بضغط ملفات React الخاصة بالواجهة الأمامية في مجلد `dist/public` وبناء ملف الخادم المجمع `dist/server.cjs`.

2. **بدء تشغيل خادم الإنتاج الفعلي:**
   ```bash
   npm start
   ```

## ✨ المميزات المضمنة في النظام

- **اتصال فوري ومباشر (WebSockets):** يربط بين الأستاذ (المعلم)، المتحكم (Host)، والطلاب بشكل فوري لتجربة مسابقة تفاعلية سريعة ومثيرة.
- **مؤثرات صوتية متطورة:** تشغيل ملفات صوتية محلية مخصصة للإجابات الصحيحة والخاطئة وحالات تفوق الطلاب الحماسية.
- **واجهة مستخدم محسّنة:** مصممة للعمل بكفاءة وبسرعة فائقة حتى على الهواتف والأجهزة ذات المواصفات المحدودة.
